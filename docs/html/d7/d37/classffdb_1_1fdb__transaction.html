Detailed Description

RAII object encapsulating a FDBTransaction If not committed, transaction is rolled back at destruction time.

Efficiently retrieve a full (depending on the potential limitation in the given option) range following the provided options.

In the case the full list is not wanted, consider using Iterators (free_fdb::make_iterator) which are more convenient to use and would save cpu/memory usage.

Parameters
fromkey from where to start the range (inclusive/exclusive depending on options)
tokey to end the range selection (inclusive/exclusive depending on options)
optadditional options for selection (limit / inclusion / exclusion etc..)
Returns
range found from the foundation db respecting the provided options.
