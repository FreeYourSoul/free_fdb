\doxysubsection{Detailed Description}
R\+A\+II Object representing an instance of the foundationdb, At construction time a thread is launched in order to handle the fdb network. At the first initialization of the foundationdb (ensured by a std\+::once\+\_\+flag semaphore) network is setup. 

Encapsulate a F\+D\+B\+Database pointer, connection to the database and network thread is closed when the object is destructed
